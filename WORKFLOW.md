## ğŸ“± User Authentication Flow

```mermaid
sequenceDiagram
    participant App as Mobile App
    participant API as Express Server
    participant DB as MongoDB Atlas
    
    App->>API: POST /api/auth/signup {name, email, password}
    API->>API: Hash password with bcrypt
    API->>DB: Save user (isVerified: false)
    API->>API: Generate 6-digit OTP
    API->>DB: Save OTP (expires in 10 min)
    API->>App: Return {userId, email, otp}
    
    Note over App: User enters OTP
    
    App->>API: POST /api/auth/verify-otp {email, otp}
    API->>DB: Find OTP record
    API->>API: Validate OTP & expiration
    API->>DB: Update user (isVerified: true)
    API->>DB: Delete used OTP
    API->>API: Generate JWT token
    API->>App: Return {token, user}
    
    Note over App: Store token in SecureStore
    
    App->>API: POST /api/auth/login {email, password}
    API->>DB: Find user by email
    API->>API: Compare password with bcrypt
    API->>API: Generate JWT token
    API->>App: Return {token, user}
    
    Note over App: Token stored for future requests
    
    App->>API: PUT /api/user/profile (with Bearer token)
    API->>API: Verify JWT token
    API->>DB: Update user profile
    API->>App: Return updated user
```
